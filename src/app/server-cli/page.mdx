# Server

The server CLI is a powerful tool that allows you to interact with the server and its functionalities natively without
the need for the GUI control panel. This guide will cover the most common commands and how to use them. {{ className: 'lead' }}

---

## Make operator {{ tag: 'CLI', label: 'make operator' }}

<Row>
    <Col>

        Create a new operator user.

        ### Positional arguments

        <Properties>
            <Property name="username" type="string, required">
                The username of the new operator, must be unique.
            </Property>
            <Property name="password" type="string, required">
                The plaintext password for the new operator.
            </Property>
        </Properties>

        ### Options

        <Properties>
            <Property name="-d, --debug" type=" ">
                Increase the verbosity of the output.
            </Property>
            <Property name="-c, --config" type="string">
                The path to the configuration file to use, relative to the current working directory.

                Default value: `config.json`
            </Property>
        </Properties>

    </Col>
    <Col sticky>

        ```bash {{title: "Command synopsis"}}
        rs2-server make operator [options] [username] [password]
        ```

    </Col>
</Row>

---

## Make jwt {{ tag: 'CLI', label: 'make jwt' }}

<Row>
    <Col>
        Generate a jwt key for the server to use for signing and verifying jwt tokens, the format is a 64 character string valid for HS512.
    </Col>
    <Col sticky>

        ```bash {{title: "Command synopsis"}}
        rs2-server make jwt
        ```

    </Col>
</Row>

---
